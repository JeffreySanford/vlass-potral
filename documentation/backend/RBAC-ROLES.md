# RBAC Roles (MVP)

Status date: 2026-02-07

## Roles
- Anonymous: read-only
- Verified User: create/edit posts and revisions
- Moderator: post moderation controls
- Admin: user and system operations

## MVP Permissions
- Read posts: all roles
- Create/edit posts: verified+
- Moderate posts: moderator+
- Manage system settings/users: admin

## Deferred Permissions
- Comment-specific permissions (v1.1)
- Mode B/FITS controls (v2)
