# API Versioning (MVP)

Status date: 2026-02-07

## Policy
- Keep a stable v1 route surface for MVP scope.
- Breaking changes require changelog + docs update.
- DTO compatibility is anchored by `libs/shared/models`.

## Deferred Domains
- Comment endpoints in a post-v1.0 increment.
- Mode B/FITS proxy endpoints in future major versions.
