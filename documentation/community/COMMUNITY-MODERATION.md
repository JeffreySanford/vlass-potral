# Community Moderation (MVP)

Status date: 2026-02-07

## MVP Moderation Scope

- Public read of published posts

- Verified write for posts/revisions

- Moderator actions on posts:

  - hide/unhide

  - lock/unlock

- **Moderation Dashboard**:

  - Centralized report triage for posts and comments.

  - Resolution workflow for community flags.

  - Real-time content lifecycle updates.

## Not in MVP

- Automated AI content filtering (v1.1)

- User reputation scoring (v1.1)
