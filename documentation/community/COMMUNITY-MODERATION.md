# Community Moderation (MVP)

Status date: 2026-02-07

## MVP Moderation Scope
- Public read of published posts
- Verified write for posts/revisions
- Moderator actions on posts:
  - hide/unhide
  - lock/unlock

## Not in MVP
- Comment moderation workflow (v1.1)
- Comment flag queues (v1.1)

## Audit
All moderation actions must be audit logged with actor and timestamp.
